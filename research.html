MSc Research

For my Master's thesis, I designed and implemented a simple scripting language to help facilitate the creation of motion controllers for physically simulated humanoid characters.

I've done some initial work on porting the system to javascript, so anyone can play around with it in a web browser. The source code for that can be found here. I've used this system to implement the SIMBICON walk controller in the browser.
