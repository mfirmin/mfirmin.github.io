Raycasting based 3D Function Grapher using GLSL Shaders

Below is an example of a 3D function graphed using a ray-casting approach. It uses GLSL shaders and Three.js in order to create an interactive, high quality surface. You can rotate the plot with the mouse, and zoom with the scroll wheel. A blog post on this, as well as a more interactive version that supports user-provided functions is coming soon! The function being graphed is the Clebsch Diagonal Cubic.
